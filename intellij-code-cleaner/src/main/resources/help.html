1. Do I need Code Climate subscription to use this plugin?

No, you don't need any subscription to use this plugin, all you need is the Code Climate CLI installed in your local machine.

2. Can I use it with my private repositories?

Yes, you can use it with public/private repositories

3. Does the plugin send my source code to anywhere?

HELL NO, all processes run on your local machine only, the plugin does not send your source-code/data to anywhere.
You can checkout source code of the plugin in here.

4. How many programing languages are supported?

Currently the plugin supports: Ruby, Python, PHP, JavaScript, Java, TypeScript, GoLang, Swift, Scala, Kotlin and C#

5. How to install Code Climate CLI locally

Please follow this document to install Code Climate CLI

6. Can I define some custom rules such as "lines in file", "lines in function"?

Yes, please check out the Advanced configuration of Code Climate documentation.

7. Duplications issues are so annoying, how can I fix it?

You can increase threshold of Duplication engine, please follow this document.
